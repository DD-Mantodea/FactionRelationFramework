## 📖 介绍
`Faction Relation Framework` 是一个给mod开发者提供的自定义新派系关系并设置关系的框架。

此mod处于刚刚开始开发的阶段，功能少，bug多都是正常现象，请积极上报来帮助作者开发。

#### ✔️ 已经完成的内容

`> 自定义关系名称，好感度范围，颜色及显示的数值等`

`> 设置自定义关系的原版关系偏向来适配原版及其他mod的逻辑，如附庸关系偏向于盟友`

`> 根据自定义关系设置交易时购买及出售的价格乘数`

`> 给通讯台添加新的选项及修改原版的选项`

`> 添加和修改定居点的浮动选项`

#### ➕ 正在开发的内容

`> 更多可扩展的钩子`
